# TORCS-QLearning

Proyecto de Aprendizaje por Refuerzo donde entrenamos un coche en TORCS utilizando Q-Learning para controlar:

- Marchas (Gear)
- Volante (Steering)
- Acelerador (Accel)
- Freno (Brake)

El objetivo es que el agente aprenda a conducir de forma autónoma maximizando la velocidad y estabilidad en pista sin salirse ni colisionar.
